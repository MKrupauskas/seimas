# Prerequisites

1. [python-3.12.1](https://www.python.org/downloads/release/python-3121/)

# Setup

```
python -m venv venv
source ./venv/Scripts/activate
pip install -r requirements.txt
```

# To run web app
```
python app.py
```

# How to deploy?
```
git push heroku main
```
